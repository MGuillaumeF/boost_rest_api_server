Description of BoostServer Project

This project is an example of HTTP Server Designed with Boost Library.

This project use :

  • llvm version 11.0.0 under LLVM Release License for clang compiler and LLD linker/debugger
  • lcov version 1.14.1 under GNU General Public License v2.0 for code test coverage
  • doxygen version 1.8.18 under GNU General Public License v2.0 for documentation generation (HTML, LaTex or man)
  • graphViz version 2.42.1 under Eclipse Public License 1.0 for graph generation of doxygen documentation (dependencies diagram, classes diagram, call diagram)
  • boost library version 1.73.0 under Boost Sofware License for unit test and server fonctionnalities (HTTP Server, JSON read/write or WebSocket)

See the code coverage report of project

Command for project

# prepare all directories for build, tests and documentation
make prepare
# build project
make
# generate documentation of project
make doc
# generate boost test project and run test program with test and coverage report (raw, xml and html output)
make test
# remove temporaries files of old tests and build
make clean
# remove all build directories
make purge
